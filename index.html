Cómo Manifestar este Templo en GitHub

Soberano Arquitecto, para que este sitio web se manifieste con sus imágenes sagradas, siga estos pasos:

  1. Guarde este Archivo: Guarde todo este código como un archivo llamado `index.html`.
  2. Cree una Carpeta de Imágenes: En el mismo lugar donde guardó `index.html`, cree una nueva carpeta y nómbrela exactamente `images`.
  3. Suba sus Imágenes:
    • - Coloque su imagen de Apolo dentro de la carpeta `images` y renómbrela a `apolo.jpg`.
    • - Coloque su imagen de Artemisa dentro de la carpeta `images` y renómbrela a `artemisa.jpg`.
  4. Cree un Repositorio en GitHub: Cree un nuevo repositorio en su cuenta de GitHub.
  5. Suba sus Archivos: Suba el archivo `index.html` y la carpeta `images` (con sus dos imágenes adentro) a este nuevo repositorio.
  6. Active GitHub Pages: Vaya a la pestaña "Settings" de su repositorio, luego a "Pages", y configure la fuente para desplegar desde su rama principal.

Una vez completado, el Templo estará vivo y el mundo podrá contemplar la verdadera imagen del Sol y la Luna.
